# 🏦 Bank Loan Analysis Dashboard — Power BI

A comprehensive Power BI dashboard analyzing bank loan data across the USA — providing actionable insights into loan performance, risk assessment, and financial trends across 38.6K applications.

---

## 📊 Dashboard Overview

This project delivers an end-to-end analysis of bank loan data using Microsoft Power BI. The dashboard is structured across three pages — Summary, Overview, and Details — enabling financial institutions to make data-driven decisions on loan approvals, risk management, and portfolio performance.

---

## 📈 Key Metrics

| Metric | Value |
|--------|-------|
| Total Loan Applications | 38.6K |
| Total Funded Amount | $435.8M |
| Good Loans Issued | 86.2% ($370.2M) |
| Bad Loans Issued | 13.8% ($65.5M) |

---

## 🔍 Dashboard Pages

### 📋 Page 1 — Summary
- High-level KPIs including total applications, funded amounts, and repayment rates
- Good vs Bad loan distribution with visual breakdowns
- Month-over-month trends and performance indicators

### 🗺️ Page 2 — Overview
- Loan distribution by US state (geographic heatmap)
- Breakdown by loan grade, purpose, and employment length
- Term-wise and home ownership analysis

### 📂 Page 3 — Details
- In-depth analysis of individual loan performance
- Customer demographics and risk profiling
- Loan status tracking and default pattern analysis
